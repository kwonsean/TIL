# 정리

과제를 하긴 하는데 깔끔한 코드가 나오질 않았다
팀회고 시간을 통해서 어떻게 하면 좀 더 깔끔한 코드를 만들 수 있는지 많이 배운 시간이었다.
문제는 이렇게 코드를 짜기 위해선 고차 배열 메서드 같이 고급스러운 스킬을 사용해야 하는데 아직까지 이해를 100%하지는 못했기 때문에 쉽지 않다.
자주 보고 사용하면서 능숙하게 사용 할 수 있도록 해야겠다.
